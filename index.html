I strive to write code that is stylistically clean, clearly and purposefully presented, concise, and as fast and efficient as possible. Alternatively, I enjoy the challenge of "code golfing" my solutions and learning clever techniques to achieve shorter code. I'm often on CodeFights.com competing in daily code golf challenges for a spot on the leaderboards.
